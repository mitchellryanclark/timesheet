# Project Context: Antigravity Timesheet App

## 📋 Overview
A professional, voice-integrated Progressive Web App (PWA) designed for tracking work hours and generating high-quality PDF reports.

## 🛠 Features
- **Voice Control:** Log shifts and adjust times using natural speech.
- **PWA Support:** Installable on mobile devices with offline capabilities.
- **PDF Generation:** Automated creation of centered, professional PDF invoices.
- **Data Sync:** Real-time synchronization with Google Sheets.
